<p>
  『스프링 게시판』은 Spring 프레임워크를 활용하여 개발된 『게시판 CRUD 』기능을 구현한 프로젝트입니다.<br>
  이 프로젝트는 『Oracle DB』와의 연동을 통해 사용자가 게시글을 『생성』, 『수정』, 『삭제』할 수 있도록 구현되었습니다.<br>
  또한 『페이징』 기능도 구현되어 있어, 페이지마다 정해진 개수의 게시물만 사용자에게 보여지도록 설계되었습니다.
</p>
